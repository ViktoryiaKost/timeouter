Privacy policy

Personal data protection policy

Innowise Sp. z o.o. (d/b/a Innowise Group) recognises that privacy is an important element of our society, so we want to explain how the app “Timeouter” (created by Innowise Group) works in a way of processing personal data. Processing personal data “Timeouter” is designed to be used by merchants in the process of interacting with buyers. Application collects only the following personal data: store owner name, store owner email address, store owner phone number, store owner location (physical address). But in some cases such data may not be considered as personal data (for example, if the phone number belongs to a company).
We want to be honest and open, so we present the data that will be processed by Innowise Group and Shopify using the application. Shopify will collect the following data: - merchants data: shop name, shop link, date of creation store, date of addition, date of update; - number of app installs. Innowise Group will collect and store the following data: - merchants data: shop name, shop session id, shop state, date of addition, date of update, store owner name, store owner email address, store owner phone number, store owner location (physical address); - other data: script title, script description, script discount code, script timer duration, script theme selector, script status selector, script date of addition, script date of update. Consent for processing When the user installs the application, he will give a permission to process his personal data specified above. Also, when installing the application, the user gives permission to access information about the owner of the store, as well as to write/read/delete/edit script tags through the script tag scope. Purpose of the processing
All data (including personal data) will process for complete and accurate identification of application users and for properly functioning of the application. Terms of processing Data is stored the moment a user uses the application. Timeouter data is deleted immediately after the modal window is deleted, through the application settings. When the application is deleted, all information about the user and the application settings is irretrievably deleted. Personal data’s processing rights The user has the right to revoke his permission to the processing of personal data at any time by deleting the application or by contacting us. Also the user has the following rights in accordance with the General Data Protection Regulations (GDPR): - right to access by the data subject (art. 15 of GDPR); - right to rectification (art. 16 of GDPR); - right to erasure, or “right to be forgotten” (art. 17 of GDPR); - right to restriction of processing (art. 18 of GDPR); - right to data portability (art. 20 of GDPR). Final provisions
